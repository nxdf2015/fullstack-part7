# fullstack-part4

## api 

 * get /:id  :  get a blog
 * delete /:id  : remove a blog
 * get /  :  get all blog
 * post / :  create a blog
 * put /:id  : update a blog
 * patch /:id/:likes : update number of like of a blog



 ## .env  
 *must include*
- MONGODB_URI  
- MONGODB_URI_TEST  
- PORT=3001
- SECRET  *use by jsonweb token*
